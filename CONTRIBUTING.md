# Contributing to NumCompute

Thank you for your interest in contributing to **NumCompute** project
We welcome contributions from everyone!

---

## How to Contribute??

### 1. Fork the Repository

Click the **Fork** button on GitHub.

---

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/NumCompute.git
cd NumCompute
```

---

### 3. Create a New Branch

```bash
git checkout -b feature/your-feature-name
```

---

### 4. Make Changes

You can:

* Add new features
* Fix bugs
* Improve performance
* Improve documentation

---

### 5. Install Dependencies

```bash
pip install -e .
```

---

### 6. Run Tests

```bash
pytest
```

Make sure all tests pass before submitting.

---

### 7. Commit Your Changes

```bash
git add .
git commit -m "Add: short description of changes"
```

---

### 8. Push to GitHub

```bash
git push origin feature/your-feature-name
```

---

### 9. Create a Pull Request

Go to your fork and click **New Pull Request**.

---

## Reporting Issues

* Use the **Bug Report** template
* Provide clear steps to reproduce
* Include error logs if possible

---

## Suggesting Features

* Use the **Feature Request** template
* Clearly explain the use case and benefit

---

## Code Guidelines

* Write clean and readable code
* Use meaningful variable names
* Follow Python best practices
* Keep functions modular

---

## Testing

* Add tests for new features
* Ensure existing tests still pass

---

## Code of Conduct

* Be respectful
* Collaborate openly
* Help others learn

---

## Need Help?

* Check the README.md
* Open a discussion

---

Thanks for contributing ❤️
