## 🌟 Project Overview

**NumCompute** is a lightweight machine learning framework built entirely using **NumPy**, designed to simulate core functionality of libraries like scikit-learn.

This project focuses on:
- Deep algorithmic understanding  
- Numerical computation  
- Clean software engineering  
- Fully vectorised implementations  

---

## 🚀 Features

- 📥 **CSV Loader** with missing value handling  
- 🔄 **Preprocessing Tools** (Scaler, Encoder, Imputer)  
- 🔍 **Sorting & Searching Algorithms** (Top-K, Binary Search, Quickselect)  
- 📊 **Statistical Functions** (Mean, Std, Quantiles, Histogram)  
- 🏆 **Ranking System** with tie handling  
- 📈 **Evaluation Metrics** (Accuracy, F1, MSE, ROC, AUC)  
- ⚡ **Optimization** (Gradient & Jacobian estimation)  
- 🔗 **Pipeline System** (like scikit-learn)  
- 🧪 **20+ Unit Tests**  
- 📊 **Performance Benchmarking**
